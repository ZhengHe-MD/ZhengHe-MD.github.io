关于我

我叫郑鹤，一名软件工程师，现居北京。本科念的是经济与金融，研究生转到计算机，做过两年算法研究，工作后把主要精力放在了服务端。习惯在陌生领域里快速调研，然后把调研变成一份报告、一次分享，或者一个真正上线的方案。

写过不少关于分布式系统、调用链追踪与存储引擎的文字，也喜欢把想法直接做成能跑起来的东西——不管是一个解释器、一个跟读工具，还是一条陪跑二十年的进度条。

我给自己定了一个很长的目标：在 2048 年前用双脚跑完 40,075 公里，也就是赤道的周长。它和写作是同一件事的两面——都相信长期的、缓慢的积累。

这个网站是我所有公开表达的唯一入口，取代了此前散落各处的博客、跑步仪表盘和简历。我把 HTML 当作第一手稿，让每一页都能承载它需要的任何交互。

如果你对其中任何内容感兴趣，欢迎通过邮件或 GitHub 找到我。
